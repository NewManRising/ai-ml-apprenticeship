1) **What is the dataset about?**
The data set contains financial information about loan applicants and whether they would be a good or bad fit for a loan.
2) **What is the target variable?** The target variable is "risk", indicating if they are suitable for getting a loan. 
3) **How many rows and columns exist?** There are 1000 rows and 10 columns (1000, 10).
4) **Are there missing values?** Yes, there are 183 missing from the saving_accounts column and 394 missing from the checking_account column. These will be handled during preprocessing. 
5) **What features are categorical?** Categorical features are sex, housing, saving accounts, checking account, and purpose.
